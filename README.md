# 第4章 - 数据可视化交互平台

这是一个基于第4章.ipynb中可视化图片创建的交互式网页应用，将Python matplotlib的可视化代码转换为网页交互体验。

## 🌟 功能特色

### 📈 5种交互式图表
1. **图书采购柱状图** - 两个地区图书采购情况对比
2. **汇率走势折线图** - 2017年与2019年汇率对比
3. **温度变化趋势图** - 最高和最低温度变化趋势
4. **科赫雪花分形图** - 基于分形几何的雪花生成器
5. **数据标记样式** - 多种标记样式和文本标注效果

### 🎮 交互控制功能
- **实时调整**：通过滑块、选择器、复选框等控件
- **数据动态更新**：所有图表支持实时数据调整
- **样式定制**：颜色、标记、线型等可视化参数
- **响应式设计**：适配不同屏幕尺寸

### 🎨 可视化特性
- 基于Plotly.js和D3.js的现代可视化库
- 流畅的动画效果和过渡
- 丰富的标记样式和颜色方案
- 专业的图表布局和标注

## 🚀 快速开始

### 方式一：直接打开网页
1. 双击打开 `index.html` 文件
2. 或使用Live Server等工具启动

### 方式二：本地服务器（推荐）
```bash
# 使用Python启动本地服务器
python -m http.server 8000

# 或使用Node.js
npx serve .
```

然后访问：http://localhost:8000

## 📊 图表功能介绍

### 1. 图书采购柱状图
- **功能**：两个地区不同类别图书采购对比
- **交互**：
  - 地区数据调整滑块
  - 堆叠/分组柱状图切换
  - 实时数据更新

### 2. 汇率走势折线图
- **功能**：美元/人民币汇率时间序列分析
- **交互**：
  - 显示/隐藏不同年份数据
  - 网格线显示控制
  - 平滑曲线切换

### 3. 温度变化趋势图
- **功能**：周温度变化趋势分析
- **交互**：
  - 最高/最低温度显示控制
  - 多种标记样式选择
  - 温度区域填充效果

### 4. 科赫雪花分形图
- **功能**：基于分形几何的雪花生成
- **交互**：
  - 迭代次数控制（0-5次）
  - 颜色自定义选择
  - 旋转动画效果
  - 多种填充模式

### 5. 数据标记样式
- **功能**：展示不同标记样式和文本标注
- **交互**：
  - 标记类型选择
  - 标记大小调整
  - 颜色自定义
  - 文本标注样式

## 🛠️ 技术栈

- **前端框架**：原生HTML5 + CSS3 + JavaScript
- **可视化库**：
  - Plotly.js - 专业图表绘制

## 🔧 自定义配置

### 修改图表数据
在 `script.js` 文件中找到对应的图表函数，修改数据数组：

```javascript
// 例如修改温度数据
const maxTemps = [28, 30, 32, 29, 27, 25, 26]; // 最高温度
const minTemps = [18, 20, 22, 19, 17, 15, 16]; // 最低温度
```

### 添加新图表
1. 在 `index.html` 中添加新的标签页结构
2. 在 `script.js` 中实现对应的图表函数
3. 更新 `showTab()` 和 `initializeChart()` 函数
